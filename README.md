## Project Overview
This project is an end-to-end data analytics solution designed to track, analyze, and optimize e-commerce sales performance. It transforms raw sales data into actionable business insights through a premium, interactive web dashboard.

## Key Features
*   **Data Processing:** Automated cleaning pipeline to handle missing values, standardize formats, and merge multiple datasets.
*   **Interactive Dashboard:** Built with **Dash & Plotly**, featuring a modern "Dark Slate" UI with Glassmorphism effects.
    *   **Live Filters:** Filter data by Month, Category, and Region.
    *   **Real-time KPIs:** Track Total Revenue, Order Volume, and Avg. Order Value.
*   **Business Intelligence:** Dedicated "Insights" module providing strategic recommendations (e.g., reducing cancellation rates, seasonal optimization).

## Tech Stack
*   **Python:** Core logic and data manipulation.
*   **Pandas:** Data cleaning and aggregation.
*   **Dash & Plotly:** Web application framework and interactive visualizations.
*   **Dash Bootstrap Components:** Responsive and professional UI design.

## How to Run
1.  Install dependencies: `pip install pandas dash plotly dash-bootstrap-components`
2.  Run the application: `python app.py`
3.  Open browser at: `http://127.0.0.1:8050/`
